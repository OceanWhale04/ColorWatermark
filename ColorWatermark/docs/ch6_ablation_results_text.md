# 八组消融实验结果叙述（与 `experiments/output/*.csv` 一致，可直接粘贴）

**说明**：以下数值均取自当前工程输出 CSV 的 **MEAN** 或表内汇总行；运动模糊实验五、六已与 **`WatermarkSystem` 维纳分支一致**，对待检测图与原始载体 **同步** 做频域维纳复原（域对齐）。低光照实验二、三、八中 CLAHE 对 **I′ 与 I_orig** 亦采用域对齐。嵌入强度 **α = 0.05**，水印为 **32×32** 二值图，载体经列表读入后统一为 **512×512**（与 `ch6_images.csv` 及实验程序一致）。

---

为系统评估本文所提水印编解码系统在退化场景下的性能，共设计八组消融实验，覆盖常规场景基线、低光照增强、运动模糊复原、参数敏感度、噪声鲁棒性及极端退化等维度。实验采用控制变量法，在其余参数固定时仅改变目标变量，以观察各预处理模块对解码准确率等指标的影响。以下逐项给出**实测结果**及简要讨论（数据为单次跑表均值，保留与 CSV 一致的有效数字）。

**实验一（常规场景基线）**  
在无色散退化的条件下，10 幅测试图平均峰值信噪比 **PSNR = 48.31 dB**，结构相似度 **SSIM = 0.99896**，解码准确率 **100%**，表明嵌入对透明性影响较小且非盲提取在无退化时可靠。

**实验二（低光照，β = 0.3，有无 CLAHE）**  
域对齐 CLAHE 后，平均准确率由 **79.57%** 变为 **79.35%**，变化 **−0.22 个百分点**；多数单幅图上两列数值相同，仅个别样本（如 Splash）出现小幅下降。CSV 中 **mean_abs_clahe_minus_low** 表明 CLAHE 确实改变了像素，但对基于最大奇异值差分的判决链影响甚微，故不宜将 CLAHE 表述为「必然大幅提升」本框架下的准确率。

**实验三（弱光衰减 β 扫描）**  
在 **β = 0.3、0.2、0.1** 下，无预处理平均准确率分别约为 **79.57%、79.56%、79.53%**；经域对齐 CLAHE 后分别约为 **79.35%、79.31%、79.34%**，相对无预处理的差值均在 **约 −0.20～−0.25 个百分点** 量级。可见在当前仿真强度与解码器下，**减弱 β 并未带来 CLAHE 收益的单调增大**。

**实验四（CLAHE clipLimit 敏感度，β = 0.3）**  
在 **clipLimit = 1.0、2.0、2.5、3.0、4.0**（tileSize = 8）下，10 幅图平均准确率分别约为 **79.36%、79.34%、79.33%、79.29%、79.04%**，整体在 **约 0.3 个百分点** 内波动，**clipLimit 过高时略降**，与限制对比度过度的直觉一致，但幅度远小于「跨十个百分点」的跃迁。

**实验五（运动模糊，L = 15，30°，有无维纳，域对齐）**  
平均准确率由无预处理的 **83.56%** 降至经双侧维纳后的 **70.35%**，差值 **−13.21 个百分点**。说明在本实现的 **DWT–DCT–最大奇异值差分** 非盲提取框架下，**频域维纳虽对齐了参考支路，仍可能扰动奇异值统计，使差分解码整体劣于仅含模糊的图像**；该结果宜在正文中如实报告并讨论（如 K 与 PSF 匹配、复原伪影对 SVD 系数的影响等），而不宜概括为「维纳必然显著提升准确率」。

**实验六（模糊长度 L 扫描，L ∈ {10, 15, 20}）**  
无预处理平均准确率随 L 增大而降低（**约 89.99% → 83.53% → 76.15%**），符合模糊加重则解码变难的预期。维纳分支在域对齐下平均准确率分别约为 **74.52%、70.23%、66.83%**；与无预处理之差分别约为 **−15.47、−13.29、−9.33 个百分点**，即 **模糊越长时，维纳相对无预处理的劣势略缩小**，但仍为负增益，与「模糊越长、复原增益越大」的简单假设并不一致。

**实验七（运动模糊退化噪声 σ 扫描，L = 15）**  
在 **σ = 1～5** 下，无预处理平均准确率由 **约 83.64%** 缓降至 **约 82.71%**；双侧维纳下由 **约 70.40%** 降至 **约 69.53%**。各噪声水平下 **无预处理均高于维纳**，差值约 **13 个百分点** 量级，表明在当前参数下，**维纳支路未表现出相对无预处理的鲁棒性优势**。

**实验八（极端退化）**  
超低光照 **β = 0.05** 时，无预处理与域对齐 CLAHE 平均准确率分别约为 **79.55%** 与 **79.34%**，变化很小。运动模糊 **L = 25** 时，无预处理与双侧维纳分别约为 **73.64%** 与 **64.39%**，维纳仍低于无预处理。若论文需强调「极端场景仍困难」，可保留定性结论，但**数值应以本表为准**。

**小结**  
八组实验在**基线与参数扫描形态**上支撑了系统可复现性；**CLAHE 与维纳在本解码器与当前参数下均未呈现对平均解码准确率的大幅正向提升**，部分场景下维纳明显劣于无预处理。撰写结论时建议：（1）明确 **域对齐** 与 **双侧维纳** 的实现口径，避免与旧版「仅复原待检图」结果混用；（2）将预处理模块表述为「可选、依赖场景与后续调参」而非「普遍显著提升」；（3）在讨论中衔接 **奇异值域特征** 与 **线性复原副作用**，为后续改进（K、PSF、自适应或替换特征）留出空间。
