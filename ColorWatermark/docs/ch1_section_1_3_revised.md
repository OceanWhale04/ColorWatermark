## 研究背景与动机（开篇段，可接在「现有水印技术…」之后）

现有水印技术对低光照、运动模糊等退化场景的适配尚不充分。深度学习方法虽在鲁棒性上具有潜力，但对算力与数据依赖较强；传统图像增强与复原算法（如 CLAHE、频域维纳滤波）实现简单、计算开销可控、无需训练，便于与水印编解码流程在工程上融合。因此，本系统采用 **DWT–DCT–SVD** 彩色图像水印框架，并串联 **CLAHE** 与 **维纳滤波** 预处理模块，形成**轻量级、可复现、可消融验证**的退化场景适配方案。

---

## 1.3 本系统的研究内容

本系统基于 **C++11/14** 与 **OpenCV 4.x** 原生 API，设计并实现一套面向低光照与运动模糊场景的彩色图像水印编解码系统。主要研究内容包括：

1. **复现并实现**基于 **DWT–DCT–SVD** 组合变换域的水印核心算法，完成彩色图像 **RGB 三通道**水印嵌入与非盲提取；在提取端对三通道得到的比特估计进行 **算术平均**，再经 **零阈值**二值化，以融合多通道冗余（实现路径见 `WatermarkCodec::extract`，**非**逐比特多数表决）。

2. **引入** CLAHE 低光照增强模块与维纳滤波运动模糊复原模块，在解码前以 **「预处理—提取」** 串联方式组织流程；通过命令行 **预处理开关**（`none` / `clahe` / `wiener`）实现模块灵活组合。对原始载体与待检测图像在启用预处理时采用 **同类型、同参数** 处理，即 **「预处理—解码域对齐」**，避免单侧增强导致的特征空间不一致。

3. **构建**常规、低光照（**β = 0.3, 0.2, 0.1**）、运动模糊（**L = 10, 15, 20**）及参数敏感度、噪声与极端退化等 **多组消融实验**，在控制变量前提下量化预处理对 **PSNR、SSIM、解码准确率** 的影响；实验表明，在本文 **最大奇异值差分** 解码器与当前参数下，**CLAHE / 维纳并不保证提升平均准确率**（部分配置下甚至出现下降），该现象在第六章结合数据讨论。

4. **遵循**软件工程规范完成可行性分析、需求分析、概要设计、详细设计、测试与性能基准等文档与脚本，形成可命令行复现的演示系统；核心与实验相关源码规模约 **两千余行**（随实验与测试程序增减略有变化）。

## 创新点（可置于 1.3 末或独立小节）

本工作的特点在于：（1）在**不依赖深度学习**的前提下，将传统 **CLAHE / 维纳** 与 **DWT–DCT–SVD** 解码链路结合，便于在资源受限环境中部署与验证；（2）提出并实现 **「预处理—解码域对齐」**策略，使非盲提取中待检支路与参考支路处于一致的预处理域；（3）通过系统消融实验，**如实报告**传统增强/复原在本奇异值域解码框架下的 **非理想增益** 现象，并从 **奇异值扰动、复原伪影、参数匹配** 等角度加以讨论，为后续改进提供可量化基线。
